## Tailwind CSS

### Add to Inline Theme

```typescript expandable
/* Defaults */
--DefaultPaddingX: 3rem;
--DefaultPaddingY: 3rem;

/* Main HTML Element */
--MainSpacing: 0rem;


/* Section */
--SectionSpacing: 3rem;

/* Items */
--ItemGapX: 2rem;
--ItemGapY: 2rem;

--InnerItemGap: 2rem;

/*  Text */
--TextGap: 2rem;
--InnerTextGap:1rem;

/* Navbar */
--NavbarHeight: 4rem;
--color-navbar: var(--navbar)
```
